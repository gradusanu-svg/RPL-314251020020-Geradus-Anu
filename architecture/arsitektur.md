# Quality Attributes

## Performance

Sistem mampu menampilkan data pasien dengan cepat.

## Security

Data pasien dilindungi menggunakan autentikasi dan enkripsi.

## Usability

Antarmuka sederhana sehingga mudah digunakan oleh petugas klinik.

## Reliability

Sistem tetap berjalan stabil saat digunakan banyak pengguna.

## Scalability

Sistem dapat dikembangkan untuk mendukung cabang klinik lainnya.
