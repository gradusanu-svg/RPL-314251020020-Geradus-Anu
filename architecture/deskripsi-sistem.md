# Deskripsi Sistem

## Nama Sistem

Sistem Informasi Klinik Kesehatan

## Tujuan

Membantu proses pelayanan pasien, pengelolaan data medis, dan administrasi klinik secara cepat dan efisien.

## Pengguna

- Admin
- Dokter
- Perawat
- Pasien

## Fitur Utama

- Login
- Pendaftaran Pasien
- Data Rekam Medis
- Jadwal Dokter
- Pembayaran
- Laporan
