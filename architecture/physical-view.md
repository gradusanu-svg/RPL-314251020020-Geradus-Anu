# Physical View

```text
+-------------+
| Client (PC) |
+-------------+
       |
   Internet
       |
       v
+------------------+
|    Web Server    |
+------------------+
       |
       v
+----------------------+
| Application Server   |
+----------------------+
       |
       v
+------------------+
| Database Server  |
+------------------+
```

## Penjelasan

Pengguna mengakses aplikasi menggunakan komputer melalui internet. Permintaan diterima oleh Web Server, diproses oleh Application Server, kemudian data disimpan pada Database Server.
