# Maintenance Plan

## Corrective

1. Memperbaiki bug pada proses login dokter.
2. Memperbaiki kesalahan penyimpanan data pasien.
3. Memperbaiki tampilan jadwal pemeriksaan yang tidak muncul.

## Adaptive

1. Menyesuaikan sistem dengan versi terbaru sistem operasi.
2. Mengintegrasikan API BPJS terbaru.
3. Menyesuaikan format laporan sesuai kebijakan Kementerian Kesehatan.

## Perfective

1. Menambahkan fitur pencarian data pasien.
2. Menambahkan fitur pengingat jadwal kontrol.
3. Meningkatkan kecepatan pencarian rekam medis.

## Preventive

1. Melakukan backup database setiap hari.
2. Melakukan pembaruan dokumentasi sistem.
3. Melakukan refactoring kode secara berkala.
